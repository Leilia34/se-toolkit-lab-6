# Task 1: Call an LLM from Code

## Выбор провайдера и модели
Я использую OpenRouter, так как он предоставляет доступ к множеству моделей без необходимости регистрации в зарубежных сервисах. Модель: `qwen/qwen3-coder:free` (или `qwen/qwen3-4b:free` в случае лимитов). API-ключ получен и сохранён в `.env.agent.secret`.

## Архитектура агента
Агент представляет собой простой Python-скрипт, который:
- Принимает вопрос из командной строки.
- Читает конфигурацию из `.env.agent.secret`.
- Выполняет HTTP-запрос к OpenRouter API.
- Возвращает JSON-ответ с полями `answer` и `tool_calls` (пустой массив).
- Весь отладочный вывод направляется в stderr.

## Структура файлов
- `agent.py` – основной скрипт.
- `.env.agent.secret` – конфигурация (не коммитится).
- `tests/test_agent.py` – регрессионный тест.
- `AGENT.md` – документация.
